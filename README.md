# Syoya_CGkadai
コンピュータグラフィックス特論で作成した課題  
  
Unityで作成  
対応HMD：HTCVive  
作成：Unity  
使用したアセット：  
SteamVR Plugin  
VRTK - Virtual Reality Toolkit  

## 概要

スタート地点からゴールまでViveコントローラを用いて移動してください。  
黄色いオブジェクトにコントローラを近づけてトリガーを引くことでオブジェクトに捕まることができます。

## 操作方法
左右トリガー：オブジェクトを掴む

## 動作方法

ファイル内のsyoya_CGkadai.exeファイルを実行してください。
